# 来場の振り返り
- フロントコーディングを担当、ほぼオンスケで進んで良かった。
- フォームはコーディングしながらの構築になり、はじめはデザインが固まっていなかったので、見た目の落とし所が難しかった。
- 想定外だったのは、ラベルアイコンの3点リーダー表示が難しかった事。フェーズ2でやっと完成した。
# 上期
ベネフィットLPのコーディングやLP反響質向上プロジェクトにてABテストや改修を行う。HTMLメルマガのテンプレート作成や依頼のあったメルマガを作成していた。
ビジパのページも引き受けて作成していた。
社内的には、伊藤準さん、鳥井さん、杉山さん、渡辺和貴さん、比下さん、矢部さんが辞められて寂しくなる。
渡辺さん無しになり、フロントは実質一人の体制。フロントコーディングのレビュワーが不足気味になる。
# 下期
コロナの影響で在宅勤務へシフト、週に一度ぐらいの出社へ。リモート環境でのコミュニケーションを色々と試すことに。
HTMLメルマガは、企画していたイベント自体が全て中止になり、一旦テンプレート改修が終わってペンディング状態に。
福田さん、嶌田さん、三ツ國さんがジョイン。
福田さんにビジパを徐々に覚えてもらって、手が離れるように移行していく。
